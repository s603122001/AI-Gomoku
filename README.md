# AI Gomoku


Play [Gomoku] (Five in a Row) against an AI system based on Monte Carlo Tree Search (MCTS).

### Compile and Execution

AI Gomoku requires C++11.

```sh
$ cd AI_Gomoku
$ g++ -std=c++11 main.cpp Board_Function.h MCST.h
$ ./ a.out
```
